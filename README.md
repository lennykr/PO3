## Cursus, Opdrachten & Toepassing

Voor het vak *Professionele ontwikkeling 3* heb ik de volledige cursus (36.5u videomateriaal) [NodeJS - The Complete Guide (incl. MVC, REST APIs, GraphQL)](https://www.udemy.com/course/nodejs-the-complete-guide/) afgerond. 

### De eerste 6 taken (Monolithic App, MVC-structuur):

    1.   **The Basics** – Basis Node.js applicatie    
    2.   **Express.js** – Implementatie van het Express.js-framework    
    3.   **Navigation** – Routes & Views   
    4.   **Templating Engines** – Implementatie van Handlebars    
    5.   **Sessions and Cookies** – State in een applicatie    
    6.   **Validation** – Input validatie via Validator.js    

### Taak 7 en 8 (REST API):
In taak 7 en 8 werd er in de cursus tijd besteed aan een REST API via Node.js. Deze structuur heb ik verder verkend in het vak Angular. Daar koos ik om te werken met Node.js in de plaats van ASP.NET, op basis van de overeenkomstige hoofdstukken in deze cursus.

De GIT-repository van dit project (asket-api) werd toegevoegd als submodule.
